Installation

The recommended way to get started using one of the drivers in your project is with a dependency management system.

MongoDB Scala Driver

The Reactive Streams based Scala implementation for asynchronous stream processing with non-blocking back pressure.

Scala 2.12


<dependencies>
    <dependency>
        <groupId>org.mongodb.scala</groupId>
        <artifactId>mongo-scala-driver</artifactId>
        <version>4.3.0_2.12</version>
    </dependency>
</dependencies>



  dependencies {
      compile 'org.mongodb.scala:mongo-scala-driver:4.3.0_2.12'
  }



  libraryDependencies += "org.mongodb.scala" %% "mongo-scala-driver" % "4.3.0"

Scala 2.11


<dependencies>
    <dependency>
        <groupId>org.mongodb.scala</groupId>
